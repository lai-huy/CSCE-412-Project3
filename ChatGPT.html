Introduction

A Red-Black Tree is a type of self-balancing binary search tree. It is a data structure that allows fast search, insertion, and deletion of elements. In a Red-Black Tree, each node has an extra bit that represents the color of the node. The nodes are colored either red or black.

Properties

  • Every node is either red or black
  • The root is always black
  • Every leaf (NIL) is black
  • If a node is red, then both its children are black
  • For each node, all simple paths from the node to descendant leaves contain the same number of black nodes
